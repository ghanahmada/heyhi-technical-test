## 1. Build docker
```bash
docker build -t <IMAGE_NAME>:<IMAGE_TAG>

# Example
docker build -t heyhi-technical-test:1.0 .
```

## 2. Run docker
```bash
docker run -p 8000:8000 <IMAGE_NAME>:<IMAGE_TAG>

# Example
docker run -p 8000:8000 heyhi-technical-test:1.0
```

## 3. Access the app
```bash
https://localhost/8000
```
